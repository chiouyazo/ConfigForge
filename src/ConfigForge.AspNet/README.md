# ConfigForge.AspNet

Adds the ConfigForge configuration UI to an ASP.NET app (Open mode). Call `AddConfigForge(...)` and `UseConfigForge()`, point it at a schemas folder and a plugins folder, and the editor is served in the browser. Brings in the UI and engine transitively.

See the [ASP.NET guide](https://github.com/chiouyazo/ConfigForge/blob/main/docs/aspnet.md).

Part of [ConfigForge](https://github.com/chiouyazo/ConfigForge).
