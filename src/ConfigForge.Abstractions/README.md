# ConfigForge.Abstractions

The contracts you reference to write a ConfigForge plugin: `IPlugin`, `IPluginRegistry`, `IActionContext`, `IConfigControl`, and the supporting types. This is the only ConfigForge package a plugin needs.

See the [plugin guide](https://github.com/chiouyazo/ConfigForge/blob/main/docs/plugins.md).

Part of [ConfigForge](https://github.com/chiouyazo/ConfigForge).
