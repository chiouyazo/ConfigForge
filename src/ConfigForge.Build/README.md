# ConfigForge.Build

MSBuild targets and a source generator for building a standalone ConfigForge executable: it embeds schemas, registers compiled-in plugins without runtime reflection, and wires up a self-contained single-file publish.

See the [standalone guide](https://github.com/chiouyazo/ConfigForge/blob/main/docs/standalone.md).

Part of [ConfigForge](https://github.com/chiouyazo/ConfigForge).
